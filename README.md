## Setup

Install dependencies

```sh
$ npm install
```

## Development

```sh
$ npm start
# visit http://localhost:1234
```

## Description

바닐라 자바스크립트로 버블, 머지 소트를 시각화하였습니다. 정렬 과정을 Queue에 담아, 딤긴 순서대로 animation을 적용하는 방식으로 구현하였습니다.
